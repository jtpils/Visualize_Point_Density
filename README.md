# Visualize Point Density
## Overview
Visualize point density by using colormap "Viridis".

![img](resources/LR10.bmp)

## Usage
`./vpd [input_file] [output_file]`

### 0:RadiusSearch
***
```
Select search type ( 0:RadiusSearch or 1:NeatestKSearch ) : 0
> RadiusSearch

Set divide : 100
> 100

Clock start
time : 3237.54

Max point density : 19175
Min point density : 1
```
***
